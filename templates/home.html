Data Management

Un projet de données commence toujours par une étape d'ingénierie des données. Vous devez récolter les données et les stocker dans un format adéquat pour pouvoir les utiliser.

Je peux vous aider à...
  • Récolter des données à partir de différentes sources comme des APIs, SQL des bases de données ou des tableurs Excel.
  • Reformater et remodeler les données pour faciliter les analyses ultérieures.
  • Nettoyer les ensembles de données en éliminant les données aberrantes et aberrantes.
  • Créez des workflows de données reproductibles
  • Automatiser la création de rapports sur les données

Exemples:
  • I collected tweets containing the #surf for 9 months to study how surfers use social media. I used the twitter API and made the data collection automatic thanks to a personnal webserver and a custom R script.
  • I created a dashboard for a company called Terres Inovia. It is directly plugged on their SQL database and allows to access its information in real time.
